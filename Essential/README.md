### 1. Vue 인스턴스 - The Vue Instance
모든 Vue는 **Vue** 함수로 새 인스턴스를 만드는걸로 시작 한다.
```
var vm = new Vue({
	// 옵션
})
```
#### 인스턴스란?
> - 인스턴스는 객체를 생성하기 위해 만들어진 또 다른 객체를 인스턴스라고 부른다.
> - 인스턴스를 만들기 위해선 객체지향에서 말하는 클래스가 필요하다.
