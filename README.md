## 시작
SPA (Single Page Application)의 속도가 빠른 프레임 워크로 Vue는 각광받고 있다.
그리고 Angular와 React보다 매우 가볍고 배움 단계가 낮기 때문에 이 부분을 시작했다.
그래서 처음 목표는 상단 Vue.js 부터 익히기!!!
